需要记住的点

  1. 变量使用不需要先声明，input输入后默认为str（字符串类型），//为向下取整的意思
  2. 数据类型：int str bool Nonetype float
  3. if else 根据行前空格数判断属于哪一层
  4. 逻辑运算符：and or not (优先级 not-->and-->or)
  5. 列表：list=["aa","bb"]，list.append("cc")，list.remove("dd") 不存在时报错，max返回最大值，min返回最小值，sorted返回排序好的列表
  6. 字典：key: value contacts={"aa":"123"} 键的类型不可变，即不可使用list，可以使用元组用()代替[]，元组不可变，添加时：contacts["bbb"]="456"，若原键值存在则覆盖，判断是否存在："abc" in contacts 返回一个布尔值，删除使用del contacts["bbb"]不存在时报错，.keys()返回所有键，.value()返回所有值，.items()返回所有键值对
  7. for 变量名 in 可迭代对象 可for id tem in tem_dict.items(): for i in range(5, 10,x): i被赋值5-9，x为步长，默认为1，还可以就一个参数y，从0开始到 y-1 while 条件:
  8. format：o = """ {0}{1} "".format(a,b) ---0,1也可以使用a,b关键词
  9. 函数：def 函数名(data1,data2): return a; 作为参数时仅使用函数名，匿名函数lambda
  10. 引入模块 import xxx 或者 from aaa import bbb
  11. 类 class xxx(父类): 构造函数：def init(self,aa,bb)
  12. 文件操作：绝对路径 与 相对路径 读文件：f=open("data.text","r")，f.read(x)读x字节 或 line=f.readline()读一行 while line != "":，readlines()返回列表，f.close()，简洁写法 with open("data.text") as f: 写文件：f.write("")
  13. 异常：try:捕捉异常，except:产生异常时进行处理
  14. assert：断言 unittest 常用单元测试库
